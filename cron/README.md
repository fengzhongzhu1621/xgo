

标准的 Cron 表达式由 5 个字段组成，表示分钟、小时、日、月、星期。

```
*    *    *    *    *
|    |    |    |    |
|    |    |    |    +---- 星期 (0-6) (星期天=0)
|    |    |    +--------- 月 (1-12)
|    |    +-------------- 日 (1-31)
|    +------------------- 时 (0-23)
+------------------------ 分 (0-59)

```

* 分（Minute）：0-59
* 时（Hour）：0-23
* 日（Day of Month）：1-31
* 月（Month）：1-12
* 星期（Day of Week）：0-6（0 表示星期天）
