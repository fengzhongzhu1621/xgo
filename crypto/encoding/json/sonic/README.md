# 性能对比

| 维度           | Sonic                          | 原生 encoding/json             |
|----------------|--------------------------------|--------------------------------|
| 性能           | 快3~10倍，低内存占用            | 较慢，高内存分配               |
| 技术           | SIMD+解释器，无反射            | 反射-based，通用但低效         |
| 适用场景       | 高并发、大数据量、低延迟       | 小规模数据、兼容性优先         |
| 迁移成本       | 低（API兼容）                  | 无需迁移                       |
