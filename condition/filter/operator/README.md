# operator 字段

原子过滤规则
基础的过滤规则，表示用于对某一个字段进行过滤的规则。任何过滤规则都直接是原子过滤规则, 或由多个原子过滤规则组合而成

| 名称       | 类型                            | 必填  | 说明  |
|----------|-------------------------------|-----|-----|
| field    | string                        | 是   | 字段名 |
| operator | string                        | 是   | 操作符 | 
| value    | 不同的field和operator对应不同的value格式 | 否   | 操作数 |
